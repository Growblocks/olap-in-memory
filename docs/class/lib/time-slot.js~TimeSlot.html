    TimeSlot

    public static fromDate(utcDate: Date, periodicity: string): TimeSlot source

    Params:

    NameTypeAttributeDescription
    utcDate Date

    Date which we want to build the TimeSlot around
    periodicity string

    One of day, week_sat, week_sun, week_mon, month, quarter, semester, year

    Return:

    TimeSlot

    The TimeSlot instance of the given periodicity containing utcDate

    Example:

    let ts = TimeSlot.fromDate(new Date(2010, 01, 07, 18, 34), "month");
ts.value // '2010-01'

let ts2 = TimeSlot.fromDate(new Date(2010, 12, 12, 6, 21), "quarter");
ts2.value // '2010-Q4'

    public get periodicity: string source

    The periodicity used for this timeslot. By periodicity, we mean, the method that was used to cut time into slots.

    For instance: year, quarter, week-sat, ...

    public get value: string source

    The value of the TimeSlot. This is a string that uniquely identifies this timeslot.

    For instance: 2010, 2010-Q1, 2010-W07-sat.
